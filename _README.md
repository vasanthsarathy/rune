# Odessa

**Goal:** A Processing/p5-style creative-coding studio written in Odin — a gallery app that runs hot-reloadable "sketches" backed by a built-in `canvas` library.

**Definition of done (v1):** From a fresh clone I can build and launch the studio, create a new sketch from the in-app "New Sketch" button, edit it in my editor and see it hot-reload live, draw with the v1 core `canvas` API (shapes, color, transforms, math, random, noise, vectors, time, input), and export the current frame + a PNG sequence.
